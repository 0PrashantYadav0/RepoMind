What is RepoMind?

        
RepoMind turns a repository into a typed knowledge graph — commits, pull requests, issues, people, decisions and chat — and answers questions by walking the relationships between them. Memory is powered by Cognee's hybrid graph + vector layer.

Security

        
Every endpoint requires an API key (scopes: read < write < admin) with per-key rate limiting. Your key is stored only in this browser's localStorage and sent as X-API-Key.
